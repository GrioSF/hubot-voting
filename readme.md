# Hubot Voting Script

Vote using [Hubot](http://github.com/github/hubot)

## Installation

Move the script into hubot/src/scripts/

## Usage

### Start a vote

    hubot start vote first item, second item, third item

### Vote

    hubot vote for first item

or...

    hubot vote for 1

or...

    hubot vote first item

or...

    hubot vote 1

### Show choices

    hubot show choices

### Show votes

    hubot show votes

### End a vote

    hubot end vote
